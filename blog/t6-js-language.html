Javascript Language

Parentheses ()

Are used to exucute functions like var result = doSomething()
They can also create a messages in the console.log like console.log("This is a message.");
They can also be used equations that follow order of operation.

Brackets []

This is used to contain an array. var array = [1,3,5]

Braces{}

Are used to define the beginning and end of the function. They are also used to separate the code into parts within the function.

Single and double quotes

Are used to make a string or strings. Can use either single or double quotes it is of personal prefrence although you have to close them the same way as you started.
