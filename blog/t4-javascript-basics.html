Javascript Basics

How does JavaScript compare to HTML and CSS?

HTML and CSS is for layout and style of the site. While javascript has different functions and can be responsible for the interactive elements of a webpage as well as. It can perform repititive task and update pages without the need of refreshing the page.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is how Javascript code is organized. Loops are one of the ways that it can be used. Loops repeat a function until the condition is met. Here is an example:

Going surfing
Is there going to be a swell?
If else no, wait a day and look again(loop)
If yes, get ready


Getting Ready
Do you have your surfboard,wetsuit and towel?
If no, find all your gear.
If yes, get in the car.

Going surfing
Do you have enough petrol to get to the beach?
If no, stop and get petrol.
If yes, go straight there and surf (repeat).

Explain the difference between accessing data from arrays and objects.

Arrays are a table of data similar to a Excel spreadsheet where you have columns and rows. You can get into an array by bracket notation and it can hold strings, numbers, functions, objects arrays.

Explain what functions are and why they are useful.

Functions are blocks of code that are assigned to a variable. This is handy because you can then call that variable instead of writing out the code each time you preform that task.
